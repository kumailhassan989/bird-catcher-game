# Bird Catcher Game

Bird Catcher Game is a fun and interactive game built using HTML, CSS, and JavaScript. The goal is to click on the birds to score points before they fall off the screen. Each bird that falls reduces your health, so stay sharp and catch them all!

## Features
- Clickable birds to score points
- Health system decreases when birds fall
- Interactive and visually appealing
- Built with pure HTML, CSS, and JavaScript
- Lightweight and responsive

## How to Play
1. Open `index.html` in your browser
2. Click on the birds before they fall off the screen
3. Keep an eye on your health bar
4. Try to score as many points as possible
